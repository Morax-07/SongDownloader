### Info
- Downloads `.mp3` files from YouTube links.
- All the `.mp3` files downloaded from YouTube is saved into `download/` directory.
- Works only on **Linux**.
- Made in `bash` \(.sh\).



<br />
<br />



### Installation
You need to install `yt-dlp` and `ffmpeg` to run this script.
Open terminal and execute this commands:

<br />

```bash
sudo snap install yt-dlp
sudo apt install ffmpeg
```

<br />
<br />


### How to use 
Write the title and the link of the songs you want to download into a `.csv` file \(for example `songs.csv`\) using this format:  
  
TITLE,LINK  

Then run the script by going into its directory and executing the command:

<br />

```bash
./download_mp3.sh PATH/TO/YOUR/CSV/FILE
```
